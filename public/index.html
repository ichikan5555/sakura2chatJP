アカウント追加

Gmail設定ガイド

Step 1: Gmailの設定でIMAPを有効化

  1. Gmailにログイン → 右上の歯車 → 「すべての設定を表示」
  2. 「メール転送とPOP/IMAP」タブ → IMAPアクセスを「有効」にして保存

Step 2: Googleアプリパスワードを生成

  1. こちらをクリックしてGoogleアカウントにログイン
  2. 2段階認証が未設定の場合は先に有効化が必要です
  3. アプリ名に「さくら転送」と入力 → 「作成」
  4. 表示された16文字のパスワードをコピー（通常のGoogleパスワードとは別物です）

Step 3: 下のフォームに入力

  • ユーザー名: xxx@gmail.com
  • IMAPサーバー: imap.gmail.com
  • ポート: 993
  • パスワードモード: 手動入力に変更
  • パスワード: Step 2でコピーしたアプリパスワード
パスワード = PREFIX + (メールアドレスの先頭1文字 + 末尾1文字 + ".") + SUFFIX
例: e14suger@... → araki0404.er.junpei0822.
